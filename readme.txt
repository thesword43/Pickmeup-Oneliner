Features
Message Deletion
Pickmeup has a built-in mechanism to delete messages containing specific phrases, primarily focused on variations of "ren is a penis" using different formats, including leetspeak.

Song Reactions
Pickmeup reacts with a special emoji when certain song titles are mentioned, adding a touch of fun to your Discord server.

-pickmeup Command
receive a random pick-up line from bot

-oneline Command
receive a random one-liner (jokes)

/compwinner Command
This command allows you to assign the "Competition Winner" role to a mentioned user. The role is automatically removed on April 1st at 12 AM GMT
